### Calculator App

The Calculator App is a simple web application that allows users to perform basic arithmetic calculations. It provides a user-friendly interface with a display for the input and a set of buttons for numbers, operators, and other functionalities.


### Features

    Perform addition, subtraction, multiplication, and division operations.
    Support for multiple themes to change the app's appearance.
    Use the keyboard for input, including numbers, operators, and the Enter key to calculate results.

### Technologies Used

    React: A JavaScript library for building user interfaces.
    mathjs: A library for handling mathematical expressions safely.
    HTML/CSS: Used for structuring and styling the app's user interface.

### Installation and Usage

    Clone the repository to your local machine:

```bash
 git clone https://github.com/your-username/calculator-app.git
 cd calculator-app 
```

### Install the required dependencies:

npm install

    Start the development server:

npm start

    Open your web browser and visit http://localhost:3000 to access the Calculator App.

### Usage

    To perform a calculation, click the buttons for numbers and operators.
    Use the "C" button to clear the input field.
    Press the "Enter" key to calculate the result (or use the "=" button).
    Select a theme from the dropdown menu to change the app's appearance.
